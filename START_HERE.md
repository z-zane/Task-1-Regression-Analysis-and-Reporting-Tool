## 🎓 What The Tool Does

### ✨ Simple 5-Step Workflow

**Page 1: Upload Data**
- CSV or Excel files
- Or use demo dataset
- Preview data

**Page 2: Explore Data**
- Summary statistics
- Missing data report
- Data quality checks

**Page 3: Select Variables**
- Auto-detect variable types
- Choose dependent variable
- Choose independent variables
- Get model suggestion

**Page 4: Run Analysis**
- Choose OLS or Logit
- Click one button
- Get full results instantly

**Page 5: View & Export**
- Beautiful regression table
- Coefficient plot with CI
- Diagnostic plots
- Export to CSV/Excel/PNG

### 📈 Models Available
- **OLS Regression** - For continuous outcomes
- **Logistic Regression** - For binary outcomes

### 📊 Output Includes
- Coefficients with standard errors
- Test statistics (t, z, F)
- P-values with significance stars
- 95% Confidence intervals
- Model fit statistics (R², Pseudo R², AIC, BIC)
- Odds ratios (for logit)
- Publication-ready tables
- High-resolution plots
