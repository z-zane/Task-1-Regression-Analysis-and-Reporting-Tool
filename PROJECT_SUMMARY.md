## Technical Specifications

### Technology Stack
- **Frontend:** Streamlit 1.28.1 (web interface)
- **Backend:** Python 3.8+
- **Statistics:** statsmodels 0.14.0 (regression models)
- **Data:** pandas 2.0.3, numpy 1.24.3
- **Visualization:** matplotlib 3.7.2, seaborn 0.12.2
- **Export:** openpyxl 3.1.2 (Excel support)

### Regression Models
- **OLS:** `statsmodels.formula.api.ols()` - Full inference with t-tests
- **Logit:** `statsmodels.formula.api.logit()` - Logistic regression with z-tests
- **Confidence Intervals:** 95% default (adjustable internally)
- **Inference:** Standard frequentist approach

### Data Handling
- **Input formats:** CSV, Excel (.xlsx, .xls)
- **Missing data:** Complete-case analysis
- **Data size:** Up to 1M+ rows (limited by system memory)
- **Variables:** Unlimited (practical limit ~100)

### Output Formats
- **Tables:** CSV, Excel (.xlsx), HTML
- **Figures:** PNG (300 DPI for publication)
- **Display:** Interactive web interface

## Limitations and Future Enhancements

### Current Limitations
1. **Missing data:** Complete-case analysis only (not multiple imputation)
2. **Categorical variables:** User must manually encode as numeric
3. **Model types:** OLS and Logit only (not negative binomial, Tobit, etc.)
4. **Interactions:** Requires manual creation before importing
5. **Time series:** No auto-correlation handling
6. **Survey data:** No sampling weights

### Possible Future Enhancements
1. Multiple imputation for missing data
2. Automatic dummy variable creation
3. Additional model types (Poisson, negative binomial, Tobit)
4. Interaction term builder
5. Panel data models
6. Survey-weighted regression
7. Model comparison tools
8. Batch analysis of multiple models
9. Custom variable transformations
10. Advanced diagnostics (VIF, condition indices, etc.)

## Support and Troubleshooting

### Common Issues - Quick Fixes

| Issue | Fix |
|-------|-----|
| "ModuleNotFoundError" | `pip install -r requirements.txt` |
| "Port already in use" | `streamlit run app.py --server.port 8502` |
| "Logit fails" | Check dependent variable is binary (0/1) |
| "Plot errors" | Ensure matplotlib can display (or use Agg backend) |
| "Memory error" | Use smaller dataset or subset rows |
